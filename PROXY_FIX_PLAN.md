## 当前问题
1. ❌ Clash 自动启动失败（端口冲突）
2. ❌ IP 验证在任务日志中不够明显
3. ❌ 预览配置 UI 横向显示（应该竖向）

## 修复方案

### 1. Clash 启动逻辑优化
**问题：** 端口被占用时切换端口，导致配置不一致
**方案：**
- 启动前检测端口，如果被占用则提示用户
- 不自动切换端口，保持配置一致性

### 2. IP 验证显示优化
**问题：** IP 验证日志不明显
**方案：**
- 在任务进度中显示当前使用的节点和 IP
- 添加醒目的图标和颜色

### 3. 预览配置 UI 修复
**问题：** YAML 配置横向滚动
**方案：**
- 前端 CSS: `white-space: pre-wrap; word-break: break-all;`
- 后端返回格式化的 YAML

### 4. 端口统一
**当前配置：** 1789
**需要确认：** 主人想用哪个端口？
