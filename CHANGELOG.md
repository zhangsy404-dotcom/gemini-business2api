# 更新日志

## 2026-03-05 - 节点管理功能增强

### ✨ 新增功能

#### 1. 节点管理核心功能
- **搜索功能**：支持按节点名称和 URL 搜索
- **全选功能**：一键全选/取消全选当前筛选的节点
- **批量删除**：支持批量删除选中的节点
- **复选框**：每个节点前添加复选框，支持单独选择

#### 2. UI 优化
- **标签页导入面板**：订阅链接和 YAML 配置使用标签页切换
- **更好的错误提示**：显示详细的错误信息（HTTP 状态码、错误详情）
- **成功提示优化**：添加 ✓ 和 ✗ 图标

#### 3. 持久化功能
- 排序方式保存到 localStorage
- 代理控制状态保存到 localStorage
- 页面刷新后保持用户选择

#### 4. 其他改进
- 搜索后全选只会选择筛选后的节点
- 删除选中时只删除已选择的节点
- 改进 API 错误处理，显示具体错误原因

### 📝 修改文件

1. `frontend/src/views/Nodes.vue` - 大量功能增强
2. `frontend/src/components/ui/ToggleSwitch.vue` - 新建
3. `frontend/src/views/Accounts.vue` - 添加排序持久化

### 🚀 使用说明

1. 重新构建前端（已完成）
2. 启动服务：`python main.py`
3. 访问节点管理页面测试新功能

### 💡 功能说明

**搜索**：在搜索框输入关键词，实时筛选节点
**全选**：点击"全选"按钮选择当前显示的所有节点
**批量删除**：选中节点后点击"删除选中"按钮
**导入**：使用标签页切换订阅链接或 YAML 配置导入方式
